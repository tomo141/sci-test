# 較正ルーブリック補足

正本のレベル定義は [SKILL.md](SKILL.md) の「レベル別定義（正答率50%の基準）」表を参照。

## 専門一致（600+）の読み方

化学の問題を化学専攻の学部卒が50%で解く → 600。  
同じ問題を高校生が解く想定ではない。

## 中間帯（250, 350, 450…）の使い方

設計・議論では中間帯の定義を使う。DB保存時は最も近い50刻み（100, 200, …, 900）に丸める。150は運用上使わない。

## よくある誤り

| 誤り | 正しい見方 |
|------|-----------|
| SI単位のマイナー記号（Bq, cd）を100にする | 教科書で習う段階を見る。Bqは高校〜400 |
| 有名人物名（ニュートン）を700にする | 内容が中学〜高校なら300以下 |
| 大学用語を分野内順番だけで600にする | 修士/博士用語は800〜900 |
| 小学生定義の100に無理やり合わせる | 100は本当に小学範囲のみ。無理なら200〜300 |
